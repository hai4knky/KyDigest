ky_digest.jsは博衣こよりさんのアーカイブを、ウェブブラウザにおいてダイジェストで見るためのプログラムです。

サンプルを見れば、何となく編集の仕方はわかると思います。
    kyg_videoId, kyg_scenarioRecommendedに適当に値をセットしてください。



・分からない方へ
    editor.htmlで出力したHTMLソースを保存すると便利だと思います。
    ただし文字コードはUTF-8で保存してください！
    また、HTMLファイルはky_digest.jsと同じフォルダに作成してください。

・javascriptを10分以上学ばれた方へ
    ky_digest.jsは読み込まれたとき、kyg_videoId, kyg_scenarioRecommendedの値を読み取ります。
    その後、idがkyDigestAppのHTML要素を探し、そこにアプリケーションを作成します。

    kyg_videoIdには動画IDをセットしてください。
    kyg_scenarioRecommendedは再生したい区間を要素として持つ配列です。
    再生したい区間は、[開始時刻, 終了時刻, タイトル]という型を想定しています。時刻の単位は秒です。



・うまく動かないときにご確認ください
    ・文字化けしますか？文字コードをUTF-8にして保存してください
    ・エディターはスクリプトと同じフォルダにhtmlファイルを保存するものと想定しています。別のフォルダにしたい場合は、スクリプトの場所の指定を編集してください。
